# Recognition APP

## Requirement

- Node.js

## How you run this

After clone this project, Copy db from "db/template" to outside (db/)
  
    npm install
    node server.js

URL : http://localhost:3000/

## API list

get Walkin gate

    http://localhost:3000/api/age/walk-in

get Current Employee in room

    http://localhost:3000/api/current-employee
